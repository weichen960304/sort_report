📝 JavaScript 實現

function bubbleSort(arr) {
    let n = arr.length;
    let comparisons = 0;
    let swaps = 0;
    
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            comparisons++;
            
            if (arr[j] > arr[j + 1]) {
                // 交換元素
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
                swaps++;
            }
        }
    }
    
    return { array: arr, comparisons, swaps };
}

📚 操作原理

氣泡排序法是一種簡單的排序演算法。它會反覆地比較相鄰的兩個元素，並將較大的元素向右移動。這個過程會重複進行，直到整個陣列排序完成。

主要步驟如下：

  1. 從陣列的第一個元素開始，依序比較相鄰的兩個元素。
  2. 如果左邊的元素比右邊的元素大，則交換它們的位置。
  3. 重複上述步驟，直到陣列的最後一個元素。
  4. 對剩下的元素重複步驟 1-3，直到整個陣列排序完成。

⏱️ 時間與空間複雜度

氣泡排序法的時間與空間複雜度如下：

  • 時間複雜度：
    • 最壞情況 (Worst Case)：O(n²)（當陣列是反向排序時，需要進行最多的比較與交換）
    • 平均情況 (Average Case)：O(n²)
    • 最佳情況 (Best Case)：O(n)（當陣列已經排序完成時，只需進行一次比較）
  • 空間複雜度：O(1)（氣泡排序法是一種原地排序演算法，不需要額外的記憶體空間）

📋 完整程式碼

以下是使用 JavaScript 實現氣泡排序法的完整程式碼：

function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換元素
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr;
}
