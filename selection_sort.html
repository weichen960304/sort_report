🎯 選擇排序法教學

Selection Sort Introduction

📚 操作原理

選擇排序法是一種直觀且容易理解的排序演算法。每一輪會從「尚未排序」的區間中找出最小值，然後與目前輪次的起始位置交換，逐步把最小元素放到正確位置。

主要步驟如下：

  1. 從第 1 個位置開始，假設該位置是目前最小值。
  2. 往右掃描剩下元素，若找到更小值就更新最小值索引。
  3. 掃描結束後，將最小值與目前位置交換。
  4. 把起始位置往右移一格，重複步驟 1-3，直到排序完成。

⏱️ 時間與空間複雜度

選擇排序法的時間與空間複雜度如下：

  • 時間複雜度：
    • 最壞情況 (Worst Case)：O(n^2)
    • 平均情況 (Average Case)：O(n^2)
    • 最佳情況 (Best Case)：O(n^2)
  • 空間複雜度：O(1)（原地排序，不需要額外陣列）
比較次數
通常較多
交換次數
通常較少
穩定性
不穩定排序

📋 完整程式碼

以下是使用 JavaScript 實現選擇排序法的完整程式碼：

function selectionSort(arr) {
    const n = arr.length;

    for (let i = 0; i < n - 1; i++) {
        let minIndex = i;

        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j;
            }
        }

        if (minIndex !== i) {
            const temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }
    }

    return arr;
}

// 測試選擇排序法
let array = [64, 34, 25, 12, 22, 11, 90];
console.log("排序前的陣列:", array);
let sortedArray = selectionSort(array);
console.log("排序後的陣列:", sortedArray);

✅ 範例輸出

執行程式碼後，您應該會看到以下輸出：

排序前的陣列: [64, 34, 25, 12, 22, 11, 90]
排序後的陣列: [11, 12, 22, 25, 34, 64, 90]

💡 適用情境

  • 適合用來教學，幫助理解「每輪選出最小值」的概念。
  • 當交換成本高、比較成本低時，選擇排序通常比氣泡排序更有優勢。
  • 不建議用於大型資料排序，因為時間複雜度固定為 O(n^2)。
