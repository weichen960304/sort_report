📝 JavaScript 實現

function insertionSort(arr) {
    let comparisons = 0;
    let shifts = 0;

    for (let i = 1; i < arr.length; i++) {
        const key = arr[i];
        let j = i - 1;

        while (j >= 0) {
            comparisons++;
            if (arr[j] > key) {
                arr[j + 1] = arr[j];
                shifts++;
                j--;
            } else {
                break;
            }
        }

        arr[j + 1] = key;
    }

    return { array: arr, comparisons, shifts };
}

📚 操作原理

插入排序法會把陣列想像成「已排序區」和「未排序區」。每一輪從未排序區拿出一個元素，插入到已排序區中正確的位置，直到全部完成。

主要步驟如下：

  1. 從第二個元素開始（索引 1），把它當成 key。
  2. 向左比較已排序區的元素，若比 key 大就往右位移。
  3. 找到第一個不大於 key 的位置後，將 key 插入該位置。
  4. 重複直到最後一個元素。

⏱️ 時間與空間複雜度

  • 時間複雜度：
    • 最壞情況 (Worst Case)：O(n^2)
    • 平均情況 (Average Case)：O(n^2)
    • 最佳情況 (Best Case)：O(n)
  • 空間複雜度：O(1)（原地排序）
