📚 操作原理

            
合併排序法採用了「分而治之 (Divide and Conquer)」的策略。它將原始陣列不斷從中間分割成兩半，直到每個子陣列只剩下一個元素為止（此時視為已排序）。接著，再將這些小的子陣列兩兩合併成較大的已排序陣列，層層遞歸向上合併，最終得到完全排序的陣列。

            
主要步驟如下：

            
    
                
  1. 將陣列從中間分割成左右兩個子陣列。
    2. 
                
  2. 對左右子陣列重複步驟 1，直到每個子陣列只剩一個元素。
    3. 
                
  3. 將兩個已排序的子陣列合併成一個更大的已排序陣列。
    4. 
                
  4. 重複步驟 3，直到所有子陣列都合併完成。
    5. 
            


            
⏱️ 時間與空間複雜度

            
合併排序法的時間與空間複雜度如下：

            
    
                
  • 時間複雜度：
                    
      
                        
    • 最佳時間複雜度：$O(n \log n)$
      • 
                        
    • 平均時間複雜度：$O(n \log n)$
      • 
                        
    • 最壞時間複雜度：$O(n \log n)$
      • 
                    
    
                
    • 
                
  • 空間複雜度：$O(n)$（在合併的過程中需要額外的陣列空間來暫存資料）

📋 完整程式碼

            
以下是使用 JavaScript 實現合併排序法的完整程式碼：

            

                
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }
    
    const mid = Math.floor(arr.length / 2);
    const left = arr.slice(0, mid);
    const right = arr.slice(mid);
    
    return merge(mergeSort(left), mergeSort(right));
}

function merge(left, right) {
    let result = [];
    let leftIndex = 0;
    let rightIndex = 0;
    
    while (leftIndex < left.length && rightIndex < right.length) {
        if (left[leftIndex] < right[rightIndex]) {
            result.push(left[leftIndex]);
            leftIndex++;
        } else {
            result.push(right[rightIndex]);
            rightIndex++;
        }
    }
    
    return result.concat(left.slice(leftIndex)).concat(right.slice(rightIndex));
}

// 測試合併排序法
let array = [64, 34, 25, 12, 22, 11, 90];
console.log("排序前的陣列: ", array);
let sortedArray = mergeSort(array);
console.log("排序後的陣列: ", sortedArray);
